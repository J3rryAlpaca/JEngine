## Opening the engine
The engine runs on Python and uses **dearpygui**. To run it, open the jcreator folder inside the JENGINE folder and run `main.py`.

## Opening your project
After you open `main.py`, you're going to see a button saying "Select Project Directory". You need to select your project's folder. Note that you need to have a file named `BASE.jcode` in your folder. Otherwise, the engine will give an error. `BASE.jcode` is the main script. It runs automatically when the game starts.

## The user interface
After you load your project, the user interface should look like this:
* A "RUN BASE.jcode" button
* A "STOP GAME" button
* The stage/scene
* A menu on the right side of the screen with details about the objects, player etc.
